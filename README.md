## Description
Edge Computing을 이용한 딥러닝 기반 마케팅 타겟 검출 및 분석 시스템


## Our purpose

* 광고판 앞 보행자 Detection/Tracking 시스템 개발
* 네트워크 최적화 - Channel Pruning, Quantization
* App 개발 - Real-Time 보행자 분석 및 시각화 자료 제공
* (Option) 보행자별 맞춤 광고 추천 시스템
